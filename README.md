# 🛒 Sumit Mart

A modern Amazon-inspired e-commerce landing page built using pure HTML and CSS.

## 📌 Project Overview

**Sumit Mart** is a responsive front-end web application designed to simulate a modern online shopping experience. The project features a clean UI, product showcase section, category cards, search bar, hero banner, and responsive design suitable for desktop and mobile devices.

---

## 🚀 Features

* Modern Amazon-style user interface
* Responsive design for all screen sizes
* Product showcase section
* Category-based navigation
* Search bar UI
* Hero banner with call-to-action button
* Professional footer section
* Pure HTML & CSS (No Frameworks)

---

## 🌐 Local Setup

Clone the repository:

git clone https://github.com/your-username/sumit-mart.git


Navigate to the project:

cd sumit-mart


Open the application:

index.html


Or simply double-click the file in your browser.

---

## 🐳 Docker Deployment

Create a Dockerfile:

FROM nginx:stable-alpine

COPY index.html /usr/share/nginx/html/index.html

EXPOSE 80
```

Build Docker Image:

docker build -t sumit-mart .


Run Container:

run -d -p 80:80 --name sumit-mart-app sumit-mart


Access the application:

http://localhost
